# 02 — 计算逻辑（公式级）

> 关联模块：`js/main.js` — `calculateHybridMetrics()`、`checkStatus()`
> 原始内容来源：`spec/frontend_spec.md` §3.2、§3.3

## 一、加权计算算法

- 单个煤种权重 = 配比（成） × 0.1
- 混合指标 = Σ( 煤种指标 × 权重 )
- 分母固定为 1.0（隐含十成满配；超出十成按实际值计算，UI 提示警告）

> 示例：2 种煤，A 配比 2 成，灰分 10；B 配比 3 成，灰分 20。
> 权重：A=0.2，B=0.3
> 混合灰分 = 10×0.2 + 20×0.3 = 2 + 6 = 8.0

### 公式定义

```
设 n 种煤，第 i 种煤：
  - 配比（成）：r_i
  - 指标值：a_i (灰分), s_i (硫分), v_i (挥发分), g_i (粘结), p_i (煤价)

权重：w_i = r_i × 0.1

混合指标：
  混合灰分   = Σ(a_i × w_i)
  混合硫分   = Σ(s_i × w_i)
  混合挥发分 = Σ(v_i × w_i)
  混合粘结   = Σ(g_i × w_i)
  综合煤价   = Σ(p_i × w_i)
```

## 二、达标判定逻辑

对于每个指标（除煤价外）：
- 若 `target_min ≤ 实际值 ≤ target_max` → **达标**
- 否则 → **超标**

煤价不参与达标判定，仅展示数值。
