# 02 — 计算逻辑（公式级）

> 关联模块：`js/main.js` — `calculateHybridMetrics()`、`checkStatus()`
> 原始内容来源：`spec/frontend_spec.md` §3.2、§3.3

## 一、加权计算算法

- 单个煤种权重 = 配比（成） × 0.1
- 混合指标 = Σ( 煤种指标 × 配比 ) / Σ( 配比 )（标准加权平均，分母为实际总配比）
- 分母 = 实际总配比 `Σr_i`；`Σr_i = 0` 时混合指标全 0（除零保护）

> 示例：2 种煤，A 配比 2 成，灰分 10；B 配比 3 成，灰分 20。
> 实际总配比 = 2 + 3 = 5
> 混合灰分 = (10×2 + 20×3) / 5 = (20 + 60) / 5 = 16.0

### 公式定义

```
设 n 种煤，第 i 种煤：
  - 配比（成）：r_i
  - 指标值：a_i (灰分), s_i (硫分), v_i (挥发分), g_i (粘结), p_i (煤价)

实际总配比：R = Σr_i （R = 0 时所有混合指标为 0）

混合指标：
  混合灰分   = Σ(a_i × r_i) / R
  混合硫分   = Σ(s_i × r_i) / R
  混合挥发分 = Σ(v_i × r_i) / R
  混合粘结   = Σ(g_i × r_i) / R
  综合煤价   = Σ(p_i × r_i) / R
```

## 二、精度

所有指标（灰分、硫分、挥发分、粘结、综合煤价）与配比均精确到小数点后 2 位。计算结果在返回前四舍五入到 2 位小数，达标判定基于 2 位小数值（与显示一致）。

## 三、达标判定逻辑

对于每个指标（除煤价外）：
- 若 `target_min ≤ 实际值 ≤ target_max` → **达标**
- 否则 → **超标**

煤价不参与达标判定，仅展示数值。
